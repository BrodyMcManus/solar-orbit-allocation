# Solar Orbit Allocation

An early working draft exploring governance frameworks for allocating orbits around the Sun in large-scale space infrastructure scenarios (Dyson-swarm-type).

**Live version:** [https://bbmic.github.io/solar-orbit-allocation/](https://bbmic.github.io/solar-orbit-allocation/)

## About

This document examines how orbits around the Sun might be allocated if humanity reaches a scale where building millions or billions of satellites becomes feasible. It explores different allocation schemes, the trade-offs between them, and what governance frameworks might best serve long-term goals of resisting power concentration, maintaining transparency, and preserving amendability.

The work includes interactive visualizations built with Three.js to help non-experts understand orbital mechanics and allocation schemes.

## Status

This is a working draft that will be updated frequently. Sections on allocation schemes, actor analysis, and governance frameworks are still in development.
